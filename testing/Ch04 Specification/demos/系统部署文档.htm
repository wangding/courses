部署文档

本文档包括如下部分：

  1. 运行环境的配置
  2. 客户端的配置
  3. Web 服务器的配置
  4. 事件监视服务器的配置
  5. 邮件服务器的配置

  3. VC.net 运行库的安装：将 系统升级文件\msvcr70.dll 拷贝至 winnt\system32 文件夹下。

Web 服务器的配置

[配置数据库]

  1. 将 <DataBase> 文件夹临时拷贝到目标电脑。
  2. 通过 SQL 查询分析器读取 MobileOffice.sql 脚本，修改数据库创建路径为目标位置，执行后将创建名为 MobileOffice 的数据库。
  3. 读取 MobileOfficeAgent.sql 并执行，建立作业。
  4. 通过 sql“导入和导出数据”工具将 Configuration.txt & Users.txt 中的数据导入到数据库 MobileOffice 中的相应表中。

[配置 web 站点]

  1. 将 <MobileOffice> 文件夹复制到默认站点的 MobileOffice 目录下。

[配置主服务器组件和服务]

  1. 在 winnt\system32 下面建立 MobileOffice 文件夹。
  2. 将 <com> 文件夹中的所有文件拷贝到刚才建立的 MobileOffice 文件夹中。
  3. 双击 db.udl 配置数据库连接信息。注意：请设置为保存密码或空密码。
  4. 执行 reg.bat 注册所有组件，并注册、启动服务。
  5. 打开 控制面板/管理工具/服务，将注册的服务(SmsMsg/MOSchedule)设置为自动启动。
  6. 执行 regfcd.reg 导入注册表信息。

　

事件监视服务器的配置

  1. 将 COM\EventLogServer 文件夹拷贝到目标电脑 winnt\system32\MobileOffice 下面。
  2. 执行 reg.bat 注册并启动服务。
  3. 打开 控制面板/管理工具/服务，将服务 EvglgAlertSvc 设置为自动启动。

　

邮件服务器的配置

  1. 将 COM\ExchangeServer 文件夹拷贝到目标电脑 winnt\system32\MobileOffice 下面。
  2. 修改 sql server 中 Configuration 表中的 ExgSrvName 记录的服务器名。
  3. 执行 reg.bat 注册组件，并注册服务。
  4. 打开 控制面板/管理工具/服务，将服务 EmailNotifySvc 设置为自动启动，并修改启动用户为 Administrator。
  5. 启动服务。
