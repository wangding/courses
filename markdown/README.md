# MarkDown

Github 开源之旅第二季：MarkDown。第一季的课程中已经带着小伙伴们在开源的江湖上安家落户了，在开源的江湖中大家有了自己的名号。在前面的课程中，我已经提过 Github 开源平台上无非两种语言：计算机语言和自然语言。如果大家感觉自己编程语言学的不太好，没有关系。我们可以先从简单的开始，在这个课程中，我们给小伙伴们讲一讲自然语言文档的编写——MarkDown。
